Darién Uncharted

Created by: Rochele Pereira Gloor (she/her), Olaoluwa Oyedokun (he/him), Ethan Cossio (he/him), Mason Bates (he/him), Brennon Best (he/him), Catalina Hernandez-Ramos (she/her), Annie Sungkajun (she/her), Kristin Carlson (she/her)

A Unity walking simulator that allows the user to work with archeological sites and local community members to explore and learn about the Darién Region of Panama.
